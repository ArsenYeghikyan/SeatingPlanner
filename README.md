# Event Seating Planner

Веб-приложение (PWA) для планирования рассадки гостей на мероприятиях с генерацией печатных PDF-документов.

## Возможности

- Создание мероприятий и управление ими в личном кабинете.
- Добавление столов с названием и вместимостью.
- Добавление гостей (ФИО, контакты, заметки, категория).
- Назначение гостей за столы и контроль рассадки.
- Настройка печатного PDF:
  - фоновое изображение (JPG/PNG);
  - выбор шрифта (TTF, встроенные);
  - цвет текста (HEX);
  - размеры заголовков и основного текста.
- Получение PDF для печати или отправки.

## Пользовательский сценарий

1. Пользователь регистрируется и входит в систему.
2. Создаёт мероприятие.
3. Добавляет столы и гостей.
4. Назначает гостей за столы.
5. Переходит в раздел «Печать» и настраивает PDF.
6. Скачивает готовый PDF.

## Архитектура

Проект построен по 4-слойной архитектуре:

- **Presentation** — PWA UI.
- **API** — транспортный слой.
- **Application** — бизнес-логика.
- **Infrastructure** — доступ к данным и внешним сервисам.

### Структура решения

```
EventSeatingPlanner.sln
└── src
    ├── EventSeatingPlanner.App
    │   (Blazor WebAssembly PWA)
    ├── EventSeatingPlanner.Api
    │   (ASP.NET Core Web API)
    ├── EventSeatingPlanner.Application
    │   (Business Logic, Use Cases)
    └── EventSeatingPlanner.Infrastructure
        (EF Core, PostgreSQL, Repositories, PDF, Files)
```

## Ключевые сущности

- **Event** — мероприятие.
- **Table** — стол.
- **Guest** — гость.
- **Assignment** — рассадка гостя за столом.
- **EventPrintSettings** — настройки печати.
- **Asset** — файлы (фон, шрифт).

## Безопасность

- JWT-аутентификация.
- Все запросы привязаны к владельцу (OwnerUserId).
- Изоляция данных между пользователями.
- Проверка загружаемых файлов (тип, размер).

## Цели MVP

- Реализуемо одним разработчиком за 1–2 месяца.
- Чистая архитектура и готовность к расширению.
- Возможность дальнейшего развития: шаблоны печати, автопосадка,
  многопользовательские сценарии, SaaS-модель.

## Требования для разработки

- .NET 8 SDK.
- PostgreSQL (для слоя Infrastructure).
